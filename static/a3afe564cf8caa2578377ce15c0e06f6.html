因项目需求需要多次切换npm以及nodejs，所写出这个文章以供参考。

使用npm n进行版本切换，对nodejs的版本控制。

安装N控制器。

sudo npm install -g n

安装nodejs的最新版本

  
$ n 5.4.1  -->（安装node.js 5.4.1版本）  
  
$ n latest  -->（安装node.js最新版本）  
  
$ n stable   -->（安装node.js稳定版本）  
  
$ n rm 4.0.0   -->（删除某个版本）

如图

  
$ sudo n （用上下选择版本会车切换版本）
