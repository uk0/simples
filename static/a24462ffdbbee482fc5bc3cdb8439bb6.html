准备工具

  • DB TDengine
  • 编程语言 Golang + Python
  • 环境 Unix Linux
  • 交易所 Bitmex Huobi Binance Okex
  • 每天增量300w数据
  • 机器配置 I7-8700 + 32GB DDR4 + GTX2060 + GTX1080 (Xgboost) 磁盘 6TB（SATA6GB/s）

编译 TDengine （修改表字段限制）

yum install gcc gcc-c++ make maven  -y # centos 7

git clone https://github.com/taosdata/TDengine

cd TDengine

mkdir build && cd build

cmake .. && cmake --build .

make install
