Crate DB重启数据恢复Shards较慢解决方案

解决Crate 重启恢复Shards速度慢的问题

  • 修改crate配置文件
    • es.api.enabled: true
  • 开始处理
    • curl -XPUT http://127.0.0.1:9200/_cluster/settings -d'
  {
  	"transient" : {
  		"cluster.routing.allocation.enable" : "none"
  	}
  }'
